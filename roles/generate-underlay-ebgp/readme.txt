Input file for this role is $PWD/host_vars/{{site}}/{{inventory_hostname}}/generated-underlay-vars.yaml
Output file for this roles is $PWD/host_vars/{{site}}/{{inventory_hostname}}/underlay-ebgp.<date and time stamp>.conf.part

This role will generate followoing config:-
- P2P interfaces
- eBGp group (underlay)
- routing policy to import and export loopback subnet between eBGP peers
- routing policy to enable load balancing on forwarding table
- GR-CORE-INTF group to configure P2P interfacs settings at global level 
- GR-EDGE-INTF group is generated here but not applied yet 
- "replace:" knob is applied on each P2P interface to make sure factory default settings are overridden by config generated by this role
